# AI 숏폼 콘텐츠 자동화 솔루션

핸드메이드 브랜드 운영자를 위한 AI 숏폼(틱톡/릴스/쇼츠) 콘텐츠 자동화 MVP입니다. Next.js(App Router) + TypeScript로 구현했으며, AI 추천과 플랫폼 연동은 데모 목적의 모킹 로직으로 동작합니다.

## 주요 기능

- **콘텐츠 프로젝트**: 제품 정보 업로드 → 템플릿 추천 → 자동 편집 초안 생성 → 자막/BGM/전환 효과 수정
- **AI 추천**: 훅(오프닝) 아이디어 생성 및 원클릭 적용, 해시태그/캡션/CTA/BGM 등 마케팅 요소 추천
- **개인화 학습**: 레퍼런스 등록, 피드백(👍/👎) 기반 재추천, 브랜드 가이드(톤 키워드) 관리, 학습 데이터 초기화
- **기획 캘린더**: 월간 그리드 뷰, 기획 카드 생성/수정/삭제, 제작 상태 관리
- **성과 분석**: 기간별 추이 차트, 상위 콘텐츠, 훅/템플릿 유형별 성과, 개선 제안, CSV 리포트 다운로드

## 시작하기

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000) 에서 확인할 수 있습니다.

## 기술 스택

- Next.js 16 (App Router, Server Actions), React 19, TypeScript
- Tailwind CSS v4
- 인메모리 데이터 스토어(`lib/store.ts`) — 데모/MVP 단계로 실제 DB 연동 전 단계
- recharts 기반 추이 차트

## 참고

AI 추천(템플릿/훅/마케팅/인사이트)과 플랫폼 연동(틱톡)은 실제 모델/OAuth 호출 없이 결정적 로직으로 모킹되어 있습니다. 데이터는 서버 재시작 시 초기화됩니다.
